# Release Logs

This directory contains automated release reports generated by the MSAL Release Agent.

Each release produces a report file: `release-{version}.md`

## Report Contents

Each report includes:
- **Pre-release validation results** — package signing, build warnings, TSA, CG, SDL checks
- **Packages published** — all 7 NuGet packages with versions
- **Post-release checklist** — milestone, GitHub release, Teams notification status
- **Links** — NuGet, ADO build, TSA, CG dashboards

## Example

```
build/release-logs/
├── release-4.83.0.md
├── release-4.83.3.md
├── release-4.84.0.md
└── README.md (this file)
```
